# EduHub Backend

Backend aplikacija za onlajn platformu za učenje (kursevi, lekcije, materijali, prijave, ocene, sertifikati, notifikacije).

## Tehnologije
- Java 17
- Spring Boot 3.3.4 (Web, Data JPA, Security, Validation)
- MySQL + Flyway
- JWT autentifikacija
- springdoc-openapi (Swagger UI)
- Maven
- JUnit 5 + Mockito

## Pokretanje
mvn spring-boot:run
Swagger UI: `http://localhost:8080/swagger-ui.html`
